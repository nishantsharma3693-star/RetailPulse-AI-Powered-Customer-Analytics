# RetailPulse – AI-Powered Customer Analytics & Demand Forecasting

## Overview
RetailPulse is a Power BI analytics project that transforms retail transaction data into interactive business insights.

## Features
- Executive Summary Dashboard
- Sales Trend Analysis
- Product Performance Dashboard
- Customer Analytics
- Country Sales Dashboard
- Profit Analysis
- Inventory Risk Dashboard
- Interactive Filters

## Tech Stack

- CSV
